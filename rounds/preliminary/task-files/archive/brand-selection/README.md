# Brand Selection Archive

这个目录归档品牌选择阶段的候选扫描材料。

## 当前状态

- 初赛品牌已锁定为 **YSL Beauty**。
- 本目录中的候选品牌扫描只作为历史依据保留，不再作为当前工作主线。
- 当前活跃的品牌选择依据在 `../../analysis/ysl-beauty-selection-evaluation-zh.md` 和 `../../../decision-records/strategy-direction/decision-brand-positioning-ysl-beauty.md`。

## 归档文件

- `brand-candidate-scan-zh.md`：中文候选品牌初步扫描。
- `brand-candidate-scan.md`：英文候选品牌初步扫描。

## 使用规则

- 只有在需要复盘“为什么没有选其他品牌”时读取本目录。
- 后续 PPT、研究和方案推进默认围绕 YSL Beauty 展开。
- 如果团队决定重新打开品牌选择，必须先新增 decision record 说明原因。
