## 使用规则

- `raw/` 只放原始材料和原始笔记，不直接写 deck 结论。
- `processed/` 放已经整理成可上 PPT、可进入论证链的洞察。
- 新增外部证据必须写来源、日期、用途和可信度限制。
- 证据要服务具体页或具体判断；不要为了堆材料而收集。
- 如果证据会改变 AI 场景、8 页结构或重新打开品牌选择，另写 `decision-records/`。

## 给下一次 Codex 的接力提示

做研究时先问“这条证据支持哪一页、哪一个 so what”。目前最需要的数据不是更多 AI 新闻，而是能证明 YSL 与中国 Z 世代关系状态、品牌标签、购买/送礼/到店/会员场景、以及 AI 如何增强关系而不稀释高级感的证据。
