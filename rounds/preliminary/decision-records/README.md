# Decision Records

Use this folder to preserve important choices made during the preliminary round.

Each decision record should be short and traceable:

- Context: what question we were deciding
- Options: the main alternatives considered
- Decision: what we chose
- Rationale: why this choice is stronger
- Implications: what this changes for research, report, slides, or design

Suggested folders:

- `strategy-direction/`: brand choice, strategic thesis, AI initiative direction, growth logic
- `design-tone/`: deck style, visual language, narrative tone, typography/color references
