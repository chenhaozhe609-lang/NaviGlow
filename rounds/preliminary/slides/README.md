# Slides Folder

This folder stores slide-related outputs for the preliminary round.

## Structure

- `deliverables/`: slide files intended for sharing, presentation, export, or review.
- `information-architecture/`: page structure, story flow, slide-by-slide content plans, and deck blueprints.
- `temporary/`: working files used during slide production.
  - `temporary/files/`: temporary assets, exports, screenshots, references, or intermediate files.
  - `temporary/scripts/`: temporary scripts used for deck generation, conversion, QA, or asset processing.

## Current Working Direction

The active preliminary-round deck is based on:

- Brand: YSL Beauty
- Structure: standard 8-page body
- Core positioning: YSL Beauty as China's Gen Z high-end self-expression co-creator
